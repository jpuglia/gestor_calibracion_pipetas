# Gestor de Calibración de Pipetas

Sistema integral para la gestión y calibración de pipetas en el laboratorio de Urufarma S.A.

## Créditos
**Author:** Juan D. Puglia  
**Organization:** URUFARMA S.A.  
**Year:** 2026

## Descripción
Este proyecto permite el seguimiento de inventario de pipetas, el registro de calibraciones y la generación de reportes de cumplimiento.

## Estructura del Proyecto
- `/frontend`: Aplicación web construida con React, TypeScript y Vite.
- `/backend`: API construida con FastAPI y SQLModel.
- `/data`: Archivos de configuración y especificaciones técnicas.
